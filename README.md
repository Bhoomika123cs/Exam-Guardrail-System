# Exam Integrity System

A real-time exam monitoring system with violation detection and credibility reporting.

## Features
- Tab-switching detection with duration tracking
- Window resize monitoring (<80% screen)
- Keyboard shortcut blocking (Ctrl+C/V, F12, PrtSc)
- Idle detection (>60 seconds)
- Real-time admin dashboard with live updates
- Trust score calculation with color coding
- PDF credibility reports for each student
- Past exams section with summary downloads

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript, Chart.js
- Backend: Node.js, Express, Socket.IO
- PDF Generation: jsPDF, jspdf-autotable
