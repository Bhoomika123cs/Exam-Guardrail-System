Start the server: Open terminal in project folder and run npm start (server runs at http://localhost:3000)

Open Admin Dashboard: Go to http://localhost:3000/dashboard.html and click "Create New Exam" → Copy the generated exam code (e.g., EXAM-ABC123)

Open Student Interface: In a new tab, go to http://localhost:3000, enter any name and the exam code, click "Start Quiz"

Test violations: In student tab, try switching tabs, pressing Ctrl+C, or staying idle → watch violations appear in admin dashboard in real-time

Submit & Download: Student completes quiz → clicks "Submit" → In admin dashboard, click student name → "Download PDF Report" to get credibility report
